# Image slots

Replace these starter placeholder images with your own photos when ready.

Recommended filenames:
- hero-board.jpg
- downconverter.jpg
- amp-interface.jpg
- drone-swarm.jpg
- rocket-flight-computer.jpg

Good image ideas:
- cropped PCB photos that do not show proprietary details
- lab bench / test equipment shots
- antenna or array prototypes
- public personal project photos
- non-sensitive hardware closeups

Avoid uploading employer-owned, proprietary, export-controlled, ITAR-controlled, classified, customer-identifying, or restricted imagery.
